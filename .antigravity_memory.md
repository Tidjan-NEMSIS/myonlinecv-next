# Mémoire du Projet - MyOnlineCV (Next.js Migration)

## 📌 Contexte (2026)
Migration de l'application "MyOnlineCV" d'une stack Vanilla (HTML/CSS/JS) vers une stack moderne **Next.js 16.2.9 (App Router + Turbopack)** avec **TypeScript** et **Tailwind CSS**.
Firebase est conservé comme backend (Auth, Firestore, Cloud Storage).
Les fonctionnalités d'IA (Groq/Gemini) et la génération de QR codes/exports sont traitées côté serveur pour plus de sécurité.

## 🏗️ Architecture et Décisions Techniques
- **Framework :** Next.js 16.2.9 (App Router, Turbopack).
- **Styling :** Tailwind CSS avec intégration du `global.css` original (adapté) pour préserver le design premium (glassmorphism, couleurs, animations).
- **Gestion d'état & Auth :** Context API (`AuthContext`, `ToastContext`), Hooks personnalisés (`useCvData`).
- **Base de données :** Firebase Firestore. L'interface `CvData` contient un champ `templateId` pour supporter le multi-template (Free/Premium).
- **Sécurité IA :** Les appels IA se font via des API Routes utilisant `lib/ai-providers.ts` pour masquer les clés (Groq/Gemini).
- **Firebase Admin :** Initialisation robuste dans `lib/firebase-admin.ts` : vérifie `FIREBASE_CLIENT_EMAIL` + `FIREBASE_PRIVATE_KEY`, sinon fallback `projectId`.
- **Sécurité API :** Middleware `lib/api-auth.ts` + vérification JWT sur `/api/remove-bg` et `/api/upload-photo`.

**Règle d'ajout d'un nouveau template :** Mettre à jour les 5 fichiers ci-dessus (type, TEMPLATES, CvRoot switch, AppearancePanel filtre + switch, TemplateSelector switch, decouvrir templates + switch).

## ⚠️ Notes pour les prochains agents
1. **Templates Premium :** `ModernTemplate` et `MinimalTemplate` restent à créer dans `components/cv/templates/`.
2. **Design :** Poursuivre avec Tailwind tout en respectant l'esthétique originale.
3. **Sécurité :** Ne pas exposer `GROQ_API_KEY` ou `GEMINI_API_KEY` côté client.
4. **shadcn-compatible layer :**
   - `lib/utils.ts` — expose `cn()` (clsx + tailwind-merge).
   - `components/ui/` — répertoire pour composants UI réutilisables.
   - Dépendances ajoutées : `clsx`, `tailwind-merge`, `lucide-react`.
5. **UI/UX Pro Max Skill :** Utiliser le skill `ui-ux-pro-max` pour tout nouveau design.
6. **Règles Strictes des Modèles de CV (Templates) :**
   - **Structure à deux colonnes obligatoires** pour optimiser l'espace.
   - **Colonne Gauche** : Contact, Expertise, Informatique, Langues, Bailleurs de Fonds.
   - **Colonne Droite** : Profil Professionnel, Expérience Professionnelle, Formation.
   - **Palette** : Toujours utiliser `var(--primary)` / `var(--accent)` dans le CSS, jamais de couleurs codées en dur.
   - **Photo de Profil** : Emplacement libre selon le design.
